SECUENCIAS DE ESCAPE:
\' : Mostrar una comilla simple dentro de una asignación o cadena en la salida de información
\" : Mostrar doble comilla dentro de una asignación o cadena en la salida de información 
"" : Dentro de las doble comillas pueden usarse '' (comillas simples) y viceversa
\\ : Barra invertida, diagonal incersa, es para mostrar una barra invertida
\n : Salto de línea
\r :Retorno
\t : Tabulación
\b : Retroceso
\f : Salto de página

OPERADOR DE AUMENTO:

Ejemplo: z = a++; //PostIncremento

OPERADOR DE DECREMENTO:
-- 
Ejemplo: Let z = ++a //PreDenremento
